# 项目运行硬件环境说明

本项目的全流程开发、训练、部署均在以下低配台式机环境中完成，验证了项目在老旧设备上的可行性，打破了大模型开发需要高端硬件的门槛。

## 完整硬件配置
| 硬件类型 | 详细参数 |
|----------|----------|
| 处理器（CPU） | Intel 酷睿i3-2120，3.30GHz主频，2核4线程，英特尔第二代酷睿Sandy Bridge架构（2012年发布） |
| 主板 | 技嘉 B75M-D3V，Intel B75芯片组，MATX规格家用主板 |
| 内存（RAM） | 8.00GB DDR3，系统识别可用6.10GB（部分容量被核显共享占用） |
| 显卡（GPU） | Intel HD Graphics 2000（i3-2120自带核显），无独立显卡，共享系统内存最大可用约2GB |
| 存储设备 | 西部数据 WD5000AAKX 500GB 7200转 SATA接口机械硬盘，分为系统盘（50.78GB）+数据盘（414.98GB） |
| 系统环境 | Microsoft Windows 10 专业版 64位，版本19043，BIOS为传统Legacy模式 |

## 适配优化说明
针对以上低配环境，项目做了以下核心优化，保证全流程可稳定运行：
1.  固定依赖版本，兼容老CPU指令集，避免新版本框架不兼容问题
2.  训练阶段优化线程数、batch_size、梯度累积参数，避免内存溢出
3.  推理阶段采用单线程运行、限制上下文窗口，保证老旧设备可流畅运行
4.  采用LoRA低秩微调技术，大幅降低训练所需的硬件门槛，无需独立显卡即可完成模型微调
