# local-custom-ai-character

**基础模型：Qwen2.5-0.5B-Instruct 轻量级开源大模型**

---

## 项目简介

本项目完整实现了从 0 到 1 的自定义对话角色 AI 开发全流程：包括训练数据准备、LoRA 低秩微调、权重合并、本地离线推理、网页端交互部署。

在 PC 端完成模型微调后，最终模型可部署在 **OPPO A57t（3GB RAM，Android 6.0）** 等老旧手机上离线运行，验证了轻量级大模型在端侧低资源设备上的落地可行性。  
全流程开发与训练均在 2012 年款低配台式机上完成，无需高端显卡与高性能硬件。

---

## 核心亮点

- ✅ **完整全流程闭环**：数据处理 → 两阶段 LoRA 微调 → 权重合并 → 离线推理 → 网页交互部署
- ✅ **低配设备深度适配**：针对 i3-2120 双核 CPU、8GB 内存、无独立显卡环境全流程优化
- ✅ **低门槛可复现**：代码带完整注释与步骤说明，有基础 Python 知识即可复现
- ✅ **完全离线推理**：模型训练完成后，所有对话推理均在本地运行，无需联网
- ✅ **端侧多端部署**：支持老旧安卓手机离线运行，实现轻量化端侧 AI 对话能力

## 仓库结构说明

| 目录 | 功能说明 |
|------|----------|
| `data/` | 训练数据格式示例，可自行扩展自定义角色对话数据 |
| `finetune/` | 模型微调模块，包含训练、权重合并、命令行推理脚本 |
| `webui/` | 本地网页聊天服务，基于 Flask + llama.cpp 实现离线交互 |
| `docs/` | 开发迭代日志、硬件环境说明，完整记录项目全周期历程 |

## 环境配置

本项目依赖版本已固定，兼容老旧 CPU 环境，避免版本冲突。

### 1. 安装 CPU 版 PyTorch（老电脑/无 GPU 用户必须执行）
```bash
pip install torch==2.5.1 torchvision==2.5.1 torchaudio==2.5.1 --index-url https://download.pytorch.org/whl/cpu
```

2. 安装其他依赖

```bash
pip install -r requirements.txt
```

---

快速开始

1. 模型微调
      参考 finetune/README.md 完成两阶段 LoRA 微调与权重合并。
2. 格式转换
      将合并后的模型转换为 GGUF 格式，放入 webui/model/ 目录（转换方法见 finetune/README.md）。
3. 启动网页服务
   ```bash
   cd webui && python app.py
   ```
4. 开始使用
   · 本地访问：http://localhost:8088
   · 局域网访问：使用设备局域网 IP 打开对应地址
